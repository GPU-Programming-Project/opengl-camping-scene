# ⛺ 3D Camping Scene Implementation
**OpenGL과 Assimp를 활용한 3D 환경 렌더링 프로젝트**

## 📝 프로젝트 개요 (Project Overview)
LearnOpenGL을 기반으로 실감 나는 3D 캠핑장 환경을 구현한 프로젝트입니다.
낮과 밤의 변화, 역동적인 조명 효과, 그리고 포스트 프로세싱 기술을 활용하여 몰입감 있는 그래픽스를 구현하는 것을 목표로 합니다.

## 🛠 기술 스택 (Tech Stack)
- **Graphic API:** OpenGL 3.3+
- **Libraries:** Assimp (Model Loading), GLFW, Glad, GLM
- **Language:** C++
- **Tools:** Visual Studio, Git
